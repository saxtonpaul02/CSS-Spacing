Letter and Word Spacing

Letter-spacing

The spacing between letters. Measuring letter-spacing is done in these three ways:

  • Using pixel values. (letter-spacing: XXpx;)
  • Relative to the font. (letter-spacing: XXcm;)
  • Normal/default. (letter-spacing: normal;)

I would never recommend using 30px or 0.5cm. But maybe you like it. I used these to clearly show what letter-spacing is.

Word-spacing

The spacing between words. Just like letter-spacing, measuring word-spacing is done in these three ways:

  • Using pixel values. (word-spacing: XXpx;)
  • Relative to the font. (word-spacing: XXcm;)
  • Normal/default. (word-spacing: normal;)

Once again, I would not recommend this amount of spacing. This is just to show what it looks like.
